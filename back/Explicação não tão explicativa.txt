Backend:

Foi criado um ambiente virtual para rodar o backend 
(caso opte por não usar, não há problema. Mas toda e qualquer instalação então deverá ser feita em sua máquina), 
nele deve ser instalado as seguintes 
bibliotecas: django e django rest framework


Comando para instalar:

pip install django
pip install djangorestframework
pip install markdown  
pip install django-filter 


Após isso, entrar no diretório backend (no console), e rodar o comando: python manage.py runserver **(0.0.0.0:porta) Opcional, se rodar sem o que está em parenteses, irá rodar por padrão na porta 8000**


Sobre a API:

- para acessar os registros das pessoas: localhost:8000/pessoas

- para acessar os registros das residencias: localhost:8000/residencias


filtros:

Não foi criado nenhum filtro para os registros de residências.
Filtros sobre cidades, bairros e estados foram aplicados diretamente para as pessoas

Exemplo de url para acessar pessoas com filtros:

http://localhost:8000/pessoas/?escolaridade=7&residencia__cidade=Caxias+do+Sul&residencia__bairro=Santa+Catarina&residencia__estado=Rio+Grande+do+Sul

Filtros: 

- escolaridade

- cidade

- estado

- bairro

A url de exemplo já mostra como são passados os argumentos para os parâmetros, todos que tem
haver com a residencia (cidade, estado e bairro), seguem o padrõa: residencia__(nome do campo)
